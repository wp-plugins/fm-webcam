== Description ==

Shows WebCam picture in Sidebar with JavaScript refresh.

== Installation ==

1. Upload `fm-webcam` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Use the Widget
